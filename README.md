# Ninh Hòa Invest AI V6 – Investor Briefing AI

Bản V6 tập trung nâng app từ “đọc tin thông minh” lên “tóm tắt tác động đầu tư”.

## Điểm mới V6

- Không cần API/backend/token.
- Tin tức vẫn lấy từ nguồn công khai/RSS/Google News RSS.
- Thêm AI Briefing cho từng tin:
  - Tác động chính.
  - Khu vực có thể hưởng lợi.
  - Cơ hội cần theo dõi.
  - Rủi ro.
  - Checklist kiểm chứng.
  - Chiến lược hành động.
- Thêm “Báo cáo nhanh 60 giây” trên trang chủ.
- Nâng mục “Tác động” thành “Briefing”.
- Thêm lọc briefing theo khu vực: Ninh Hòa, Ninh Diêm, Ninh Hải/Dốc Lết, Ninh Xuân, Ninh An, Tân Định, Vân Phong, Cao tốc.
- Nâng tra cứu pháp lý: thêm gợi ý kiểm chứng và link tìm văn bản theo từ khóa.
- Lưu tin/khu vực tương thích từ V5 sang V6.

## Cách build APK bằng GitHub Actions

1. Upload toàn bộ source lên repo GitHub.
2. Vào tab Actions.
3. Chọn Build APK.
4. Đợi build xong.
5. Tải APK ở mục Artifacts.

## Ghi chú

Bản No API phù hợp cho người dùng non-tech, nhưng dữ liệu tin phụ thuộc vào nguồn công khai và mạng Internet. Với nhu cầu phát hành thương mại lớn, vẫn nên có backend riêng ở giai đoạn sau.
