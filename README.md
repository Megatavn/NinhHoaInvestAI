# Ninh Hòa Invest AI V10 – Premium UI

Bản V10 tập trung nâng cấp giao diện cao cấp hơn cho app No API.

## Điểm mới chính

- Giữ nguyên toàn bộ tính năng V8 Deal Scoring Pro.
- Nâng UI sang phong cách premium: glass card, gradient, shadow mềm, bố cục sạch hơn.
- Thêm splash screen khi mở app.
- Thêm dark mode, lưu trạng thái trên máy.
- Thêm nút nổi: làm mới tin, đổi giao diện, lên đầu trang.
- Chuyển tab mượt hơn, hiệu ứng tương tác nút/card/chip.
- Bottom navigation bo góc floating cao cấp hơn.
- Tối ưu trải nghiệm người dùng non-tech: không cần API, không cần backend, không cần cấu hình.

## Tính năng vẫn giữ

- Tin tức No API từ nguồn công khai/RSS/Google News.
- Sửa lỗi mở nguồn RSS/XML từ V7.1.
- Investor Briefing.
- Tra cứu khu vực đất chuyên sâu.
- So sánh khu vực.
- Tra cứu pháp lý và nguồn tin cậy.
- Phân tích lô đất chuyên nghiệp: điểm đầu tư, rủi ro, giá/m², vùng giá đàm phán, checklist kiểm chứng.
- Lưu tin, lưu khu vực, lưu báo cáo phân tích trên máy.

## Cách build APK bằng GitHub Actions

1. Upload/cập nhật source lên repo GitHub.
2. Vào tab **Actions**.
3. Chọn **Build APK**.
4. Đợi build xong.
5. Tải APK ở mục **Artifacts**.
