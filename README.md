# Ninh Hòa Invest AI V31 – Taste Skill & Tri thức đầu tư

Ứng dụng Android No API tập trung Ninh Hòa, tin tức đầu tư, kiểm tra lô đất, sổ tay cá nhân và mục Tri thức đầu tư.

## V31 có gì mới

- Áp dụng phong cách thiết kế từ file `taste-skill-main.zip`: mobile-native, ít nhiễu, chữ rõ, card gọn, chuyển cảnh mượt hơn.
- Thêm tab **Tri thức** để lưu bài báo/video về kinh doanh, bất động sản, pháp lý, đàm phán và tài chính.
- Cho phép người dùng tự thêm link bài báo/video từ báo chí, YouTube, Facebook, Zalo... vào thư viện riêng.
- Thêm các ý tưởng nâng cấp/góc nhìn đầu tư để gia đình dùng như sổ tay học nhanh.
- Giữ trọng tâm tin tức Ninh Hòa và các tin hạ tầng/logistics quan trọng.
- Không yêu cầu quyền vị trí, không backend, không API người dùng.
