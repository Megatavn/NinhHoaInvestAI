# Ninh Hòa Invest AI V7 – Tra cứu khu vực đất chuyên sâu

Bản V7 nâng cấp trọng tâm từ V6 Investor Briefing AI sang **Area Intelligence**: hồ sơ khu vực đất chuyên sâu cho nhà đầu tư bất động sản Ninh Hòa – Khánh Hòa.

## Điểm mới V7

- Không cần API/backend/token.
- Giữ tính năng tin tức thông minh, Investor Briefing, AI phân tích lô đất.
- Thêm tab **Khu đất** với hồ sơ chuyên sâu theo khu vực:
  - Ninh Diêm
  - Ninh Hải / Dốc Lết
  - Ninh Xuân
  - Ninh An
  - Trung tâm Ninh Hòa
  - Tân Định
  - Ninh Thủy
  - Ninh Phước
  - Ninh Ích
  - Ninh Phụng
  - Ninh Sim
  - Khu gần Vân Phong
- Mỗi hồ sơ có:
  - Điểm tiềm năng
  - Thanh khoản
  - Rủi ro pháp lý
  - Rủi ro giá
  - Khung đầu tư
  - Nhà đầu tư phù hợp
  - Động lực tăng trưởng
  - Rủi ro chính
  - Checklist pháp lý cần kiểm tra
  - Nên tránh
  - Chiến lược đề xuất
  - Nút mở tin liên quan
- Thêm so sánh 2 khu vực đất.
- Nâng AI phân tích lô đất để dùng dữ liệu hồ sơ khu vực V7.
- Có lưu khu vực/tin quan trọng.

## Cách build APK bằng GitHub Actions

1. Upload toàn bộ source lên repo GitHub.
2. Vào tab **Actions**.
3. Chọn **Build APK**.
4. Đợi build xong.
5. Tải APK ở mục **Artifacts**.

## Ghi chú

Bản No API phù hợp người dùng non-tech. Tin tức lấy từ nguồn công khai/RSS/Google News RSS nên vẫn phụ thuộc kết nối mạng và khả năng truy cập nguồn công khai. Hồ sơ khu vực dùng để sàng lọc đầu tư, không thay thế kiểm tra pháp lý thực tế tại cơ quan có thẩm quyền.
