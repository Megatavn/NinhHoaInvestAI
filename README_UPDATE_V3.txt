Bản cập nhật V3 - Ninh Hòa Invest AI

Đã sửa:
- Phông chữ/tiếng Việt có dấu trong toàn app.
- Tin tức có màn Đọc nhanh và nút Về danh sách.
- Nút Mở nguồn sẽ mở bằng trình duyệt ngoài, không kẹt trong app.
- Thêm nguồn: Tuổi Trẻ, VnExpress, PLO, Báo Chính phủ, Vietnamnet, Thanh Niên, Nhân Dân, Báo Khánh Hòa, Cổng Ninh Hòa, UBND Khánh Hòa, Công báo Khánh Hòa, IPA Khánh Hòa.
- Thêm tìm kiếm tin, lọc tin, tra cứu khu vực đất, tra cứu văn bản pháp luật.
- Nâng cấp AI phân tích đầu tư: điểm, chiến lược, rủi ro, điểm mạnh.

Cách build trên điện thoại:
1. Upload ZIP này vào Codespaces.
2. Chạy:
   unzip -o NinhHoaInvestAI_Update_V3.zip
   cp -r NinhHoaInvestAI_Update_V3/. .
   git add .
   git commit -m "Update app V3 pro news and legal lookup"
   git push
3. Vào Actions > Build APK > tải artifact APK.
