Project này đã có sẵn GitHub Actions workflow:
.github/workflows/build-apk.yml

Bạn upload project này lên GitHub, vào tab Actions và chạy Build APK.
GitHub sẽ build APK chuẩn bằng môi trường Android SDK/Gradle trên cloud.

Sau khi build xong, tải APK ở mục Artifacts.
